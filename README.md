# OMData102

## 开智学堂 数据科学入门班 简介

### 课程目标

- 面向小白同学
- 能了解一些数据分析方法
- 能用python完成常见分析任务

### 课程大纲

0. 导论
1. 工具基础（linux/ipython）
2. 数值计算（numpy）
3. 数据绘图（matplotlib）
4. 数据操作（pandas）
5. 概率和统计分析（statsmodels）
6. 数值计算（scipy）
7. 线性模型（statsmodels）
8. 机器学习初步（sklearn）
9. 毕业典礼+大作业


### 课程安排

#### 整体安排


每周课程元素：视频+幻灯+代码+练习+阅读资料+互动答疑+毕业项目

总共 9 次课，每次课程视频时长 45 mins 左右。

### 附：缩写规约

#### 数字

- 充分使用程序思想
- 数字序列从 0 开始
- 自然周从 星期天 开始,计为 `第0日`

#### 时间

- `w` ~ week 周
- `d` ~ day 日
- `h` ~ hour 小时
- 时间的表述最短为 `.5h` ~ 半小时
- 以固定的课程起点为中心点
    + 开课前,为负数
        * `-1wd3` 为 1.12 前一周第三天,即 160106
    + 开课后,为正数
        * `0wd1` 为 1.12 当周第一天,即 160111
